# Task Templates

## 1. Planning
- **Objective**: Clear statement of the goal.
- **Constraints**: Technical or business limitations.
- **Steps**: Numbered list of atomic actions.
- **Verification**: How success is measured.

## 2. Implementation
- **File**: `path/to/file`
- **Change**: `Old -> New` or `Insert/Replace` logic.
- **Logic**: Brief technical justification if non-obvious.

## 3. Verification
- **Command**: Exact command run.
- **Expected**: Expected output.
- **Actual**: Observed output.
- **Result**: PASS/FAIL.
